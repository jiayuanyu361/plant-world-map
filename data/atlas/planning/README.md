# 首批 50 条植物扩充方案

这份规划文件用于把 Plant World Map 从当前示范级植物库，扩展到更适合地图探索和演示的产品级植物图谱。

## 扩充目标

- 首批扩到 50 条高质量植物
- 优先覆盖四季变化
- 优先覆盖 4 类产品体验：
  - 观花植物
  - 观叶植物
  - 乔木 / 城市树木
  - 香草 / 水生 / 多肉
- 优先保证：
  - 名称准确
  - 地区分布合理
  - 图片真实可核对
  - 花卉类尽量补花语

## 文件说明

- `phase-1-expansion.json`
  首批 50 条植物扩充清单，包含季节、地区、代表城市、栖息环境、优先级。
- `../templates/batch-template.json`
  后续整理成可导入批次时使用的标准模板。

## 推荐执行顺序

1. 先按 `priority = high` 选图和补数据
2. 每次整理 8~12 条植物为一个 batch
3. 将 batch 放入 `data/atlas/inbox/`
4. 运行：

```bash
npm run atlas:import
```

5. 执行：
   - `supabase/atlas-catalog.generated.sql`
   - `select public.sync_approved_atlas_entries();`

## 建议的第一批优先植物

- 春季：樱花、蓝花楹、郁金香、桃花、牡丹
- 夏季：荷花、向日葵、薰衣草、茉莉、三角梅
- 秋季：银杏、桂花、日本红枫、茶树、橄榄树
- 冬季：山茶、梅花、仙人掌、芦荟、水仙

这 20 条最适合优先做成“首页可展示 + 地图可识别 + 审美稳定”的核心植物库。
